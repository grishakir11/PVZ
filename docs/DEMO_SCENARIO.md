# Сценарий демонстрации

## 1. Обзор системы

Открыть интерфейс:

    streamlit run app.py

На вкладке «Обзор» показать назначение системы, архитектуру и реализованные модели.

## 2. Просмотр отдельной модели

Перейти на вкладку «Модель», выбрать одну из моделей размещения и показать:

- карту выбранных локаций;
- таблицу выбранных точек;
- распределение по районам;
- числовую сводку.

## 3. Сравнение моделей

Перейти на вкладку «Сравнение» и показать:

- общую карту четырёх конфигураций;
- сводную таблицу;
- пересечение выбранных точек;
- районное распределение.

Основной комментарий: разные критерии оптимизации приводят к разным пространственным конфигурациям сети.

## 4. Артефакты исследования

Перейти на вкладку «Артефакты» и показать итоговые карты, таблицы и графики.

## 5. Полный pipeline

Открыть страницу «Полный расчёт pipeline» и показать, что система поддерживает запуск расчётных этапов с выбранными входными данными.
